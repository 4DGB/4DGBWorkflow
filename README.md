# Input Parameters

The input to the workflow is a yaml file with the following values:

```
project:
    name:       "Project Name"
    interval:   200000

arrays:
    - name:       "array one"
      data:       somefile.csv
    - name:       "array two"
      data:       somefile.csv

datasets:
    - name:     "Name of first dataset"
      hic:      data01.hic
    - name:     "Name of second dataset"
      hic:      data02.hic
```
